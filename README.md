## An OpenGL Reimagining of Hamster Powered Engine

**Features**
- Model Loading
- Hierarchial Transforms
- Blinn-Phong Shading
- Directional and Point Lights
- Skybox Rendering and Reflections
- Scene Serialization / Deserialization
- Static & Animated Sprites
- Cloth Simulation
- 2D and 3D Rendering
- Text Rendering
